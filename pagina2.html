Curso de HTML

Aprende HTML para crear la estructura de tus páginas web.


Lección 1: Colocar parrafos y títulos

En esta lección aprenderás lo basico, poner texto en la pagina. El poner un parrafo se hace mediante los signos "<>" juntos, esto de poner los signos de mayor y menor juntos es algo basico. Mediante estos simbolos vos entre medio pones el comando que quieras ejecutar. Para el parrafo es la letra "p" sola, pero eso es solo para abrir, para cerrar el parrafo tenes que copiar y pegar el mismo comando, pero agregando entre el < y la p, el simbolo "/"

Para los titulos y subtitulos es el mismo proceso, pero en vez de una "p" sola, ponemos h1 (solo para titulos principales, para hacer subtitulos use h2, h3 para subtitulos mas chicos, usted puede poner la cantidad de tipos de subtitulos tanto como pueda, solo debe poner la "h" con el numero del subtitulo al lado).
