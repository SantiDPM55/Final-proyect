Leccion 2: Colores y Tamaños

Para cambiar el fondo de color y el tamaño de imganes, textos y fondos es distinto.

Para el fondo es con la siguiente linea

background-color: Color que desea aqui;

Para modificar el texto es con la siguiente linea

font-size: 30px; para el tamaño, el numero 30 lo puede cambiar por el tamaño que desee de la letra

color: color deseado aqui; para el color.

Este es el comando para cambiar de tamaño las imganes

max-width: 50%; si quiere que la imagen sea mas grande o mas chica depende del porcentaje.
