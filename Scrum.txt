SPRINT:
¿Qué es lo que vamos a hacer? 
Una pagina de cursos sobre HTML y CSS. Los visitante de la pagina podran aprender sobre los antes mencionados y ademas podran reguistrarse con su email.


¿Cómo lo vamos a hacer?
* Se dialogó sobre el lenguaje y editor de texto a utilizar.
* Se realizó un boceto de la idea principal
* Se inició con el código
* Se realizó el formato en HTML, el diseño en CSS y JavaScript, imágenes, texto 
* Parte lógica, en donde se utilizó JavaScript, creando la interacción del usuario con la aplicación(Backend). Se programó el funcionamiento de botones.
* Base de datos:
* Se estableció un servidor local a través de XAMPP (herramienta de software libre para ejecutar un servidor Apache de forma local “localhost” sin la necesidad de internet) llevando a cabo la conexión por medio de php ( lenguaje de programación que permite el desarrollo web o aplicaciones web dinámicas) y dentro del mismo se adjuntó la parte de Frontend (formulario: HTML y CSS).
