o como hubo una enorme desorganización.

así que con algo de ayuda de los profes pudimos enlazarlo y crear una Base de Datos.
Respecto al uso del aula, es demasiado incómoda, requiriendo mucho tiempo para entenderla, por lo general nosotros estamos acostumbrados a su uso,
