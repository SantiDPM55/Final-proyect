Bienvenido a Cursos Webs

¡Aprende desarrollo web con nuestros cursos de html y css!

¿Quieres crear tus paginas web?

Bueno, este es el curso pensado para ello

A día de hoy cualquier empresa independientemente de su tamaño debe poseer una página web, el saber HTML y CSS es una buena forma de conseguir salida laboral.

En este curso solo aprenderas cosas basicas de ambos, luego de terminar de leer el material dejado vas a depender de ayuda externa para lo que es base de datos, uso real y práctico, razón por la cual es gratuito, debera seguir aprendiendo de forma autodidacta.

Tampoco hay certificación por la razón comentada, lo ideal es que lo tome de base para saber si quiere orientar su vida a la programación web, tampoco se le enseñara sobre paletas de colores ideales, cuando conviene usar tal color o es mejor este otro, eso forma parte de la carrera de diseñador web.

Sus paginas luego de este curso no seran profesionales, pero por su sencillez sabrá y entendera si quiere dedicarse a esto, o si lo ve como una herramienta laboral a futuro.
